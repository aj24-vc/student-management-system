

## Internship Type
Self Internship (Academic Project)

## Objective
To develop a web-based Student Management System that allows
adding and viewing student records using PHP and MySQL.

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## Features
- Add student details
- View student records
- Database connectivity

## How to Run the Project
1. Install XAMPP
2. Copy project folder to `htdocs`
3. Create database `student_db`
4. Run `index.php` in browser

## Internship Duration
4 Weeks

## Outcome
Gained hands-on experience in PHP, database handling, and web development.
